# Magic Web Ideas

This file collects future ideas for DekoKraft CMS and the Magic Web product engine. These are ideas, not implementation requirements.

## Smart Product Creation

- Convert natural-language product descriptions into structured fields.
- Detect product category, material, scent, dimensions, colors, and tags from text.
- Suggest product names in Arabic, English, German, and French.
- Generate short and long product descriptions.
- Generate SEO title, SEO description, URL slug, and image ALT text.
- Suggest collections and product types.

## Smart Image Workflow

- Convert uploaded images to optimized WebP later.
- Detect image quality and warn about blurry or dark images.
- Suggest best main product image.
- Generate social media crops.
- Generate catalog images.
- Generate background-removed product images.
- Generate AI reference variants later.

## Product Engine

- Make every pipeline step clickable.
- Show exact missing requirements for each step.
- Add completion percentage.
- Add draft autosave later.
- Add publish readiness validation.
- Add preview of customer-facing product page.

## Store And SaaS Direction

- Turn DekoKraft CMS into a reusable Magic Web SaaS.
- Support multiple stores later.
- Support creator/craft business templates.
- Support multilingual storefront content generation.
- Support product packs for candles, gifts, learning games, services, and custom orders.

## Learning And Game Modules

- Add learning product templates.
- Add memory game product modules.
- Add educational game metadata.
- Add smart age-range suggestions.
- Add printable/physical product variants.

## Future AI Integrations

- Add AI only when explicitly requested.
- Start with a dry-run/mock layer before real calls.
- Keep generated output editable.
- Show confidence and source fields.
- Never overwrite user input without confirmation.
